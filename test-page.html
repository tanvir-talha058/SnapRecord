SnapRecord Test Page

Use this page to test your SnapRecord screen recording extension

How to use this page:
  1. Install the SnapRecord extension in your browser
  2. Click the SnapRecord icon in your toolbar
  3. Select "Current Tab" to record this page
  4. Interact with the elements below while recording
  5. Stop the recording and check the downloaded video
